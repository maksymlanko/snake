Educational project to review ASM before I move on to other things :)

# Instructions
install dosbox and nasm (depends on package manager):
> sudo pacman -S dosbox nasm

add to `dosbox.conf`:
> [autoexec]
> mount c ~/dosprograms
> c:

run:
> make

Enjoy!
